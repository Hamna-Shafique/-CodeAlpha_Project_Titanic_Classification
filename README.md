# Titanic Survival Classification

Machine Learning project to predict passenger 
survival on the Titanic using classification models.

## About
Built as part of CodeAlpha Data Science internship.
Analysed Titanic dataset and developed ML model 
to predict which passengers survived.

## Tools & Technologies
Python | Pandas | NumPy | scikit-learn | 
Jupyter Notebook | Matplotlib

## What I did
- Cleaned and preprocessed Titanic dataset
- Performed Exploratory Data Analysis (EDA)
- Built classification model for survival prediction
- Evaluated model performance and accuracy
- Handled missing data and feature engineering

## Files
- main.ipynb — main Jupyter notebook
- train.csv — training dataset
- test.csv — test dataset
- gender_submission.csv — submission file

Data Cleaning | EDA | Classification | 
Feature Engineering | Model Evaluation
